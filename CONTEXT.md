# LiuYinChu Space

This context defines the product language for LiuYinChu'Space, a personal website that collects the author's writing, projects, resources, academic presence, and experiments.

## Language

**Personal Site（个人网站）**:
The whole website as an owner-directed personal home whose structure and visual character should evolve from the author's own taste.
_Avoid_: Portfolio template, blog-only site, academic CV-only site, resource directory-only site

**Growth Order（成长顺序）**:
The preferred sequence for evolving the site: visual refinement first, then new functionality, then new pages.
_Avoid_: Feature-first roadmap, page sprawl, total redesign
